# Getting Started

## Requirements:

```
Maven: 3.9+
JDK: 25
```

### Clone this repository:

```bash
git clone https://github.com/deepaksorthiya/java-maven-starter-template-latest.git
```

```bash
cd java-maven-starter-template-latest
```

### Run Test(Maven Should Be Installed):

```bash
mvn clean test
```
